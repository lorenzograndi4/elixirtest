# FizzBuzz

  FizzBuzz translates numbers (from 1 to 100) into:
  - "buzz" if it's a multiple of 3
  - "fizz" if it's a multiple of 5
  - "fizzbuzz" if it's a multiple of 3 *and* 5
  - the number itself if it doesn't match the above
